# Swing Bank (Java Swing + SQLite)

Мини-банк с ролями пользователь/админ, фиолетовой градиентной темой и локальной SQLite-базой. Авторизация по фиксированным логинам, операции пополнения/снятия и переводы на чужие счета. В комплекте раннеры: IDE, fat JAR и jpackage (app-image / exe).

## Стек
- Java 17, Swing + FlatLaf (тёмная тема)
- SQLite (jdbc, файл `bank.db` в корне проекта, создаётся при первом запуске)
- Maven + Shade + jpackage

## Демо-учётки и роли
- admin / admin — роль ADMIN, видит все счета, может корректировать балансы.
- user / pass — роль USER, личные операции и переводы другим.
- alice / pass — роль USER.
- bob / pass — роль USER.

У каждого 1–2 счета; свои счета в списке переводов не отображаются.

## Функционал
- Логин по фиксированным парам (user/pass, admin/admin, alice/bob).
- Дашборд: общая инфа, агрегаты по счетам/балансу.
- ЛК пользователя: список счетов (<3), пополнение, снятие, перевод на чужие счета.
- Админ: таблица всех счетов, корректировка балансов (+/-).
- SQLite в корне (`bank.db`), автосоздание и сид при первом запуске.

## Запуск
1. Собрать:
   ```bash
   mvn clean package
   ```
2. Из IDE: запустить `com.bank.App`.
3. Fat JAR (перетаскивает зависимости внутрь):
   ```bash
   java -jar target/swing-bank-1.0.0-shaded.jar
   ```

`bank.db` будет создан автоматически и засиден. Чтобы сбросить данные — удалите файл и перезапустите.

## jpackage / exe
- Windows-only: после `mvn clean package` выполните в терминале PowerShell/Command Prompt:
  ```
  scripts\\jpackage-win.cmd
  ```
  Появится `target/installer/SwingBank-1.0.0.exe` с вложенной runtime (jpackage сам соберёт её через jlink). Нужен JDK 17+ на Windows.
- На macOS/Linux можно запускать `java -jar ...` либо собрать свой app-image/дистрибутив аналогичной командой jpackage, поменяв `--type`.

## Структура UI
- Login → Dashboard (общая инфа) → ЛК (список счетов, +/-, перевод на чужие карты) → Admin (просмотр всех счетов, корректировка балансов).
- Фиолетовая палитра, градиентный фон, карточные панели.

## Замечания
- Транзакционная прослойка упрощена: операции мгновенно меняют баланс; перевод выполняется в одной БД-транзакции.
- Если хотите начать заново, удалите `bank.db` и соберите/запустите снова — таблицы и сиды развернутся автоматически.
